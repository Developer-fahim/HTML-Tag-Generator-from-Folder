🔥 HTML Tag Generator from Folder

Easily generate clean HTML tags for all files inside a folder – images, stylesheets, scripts, and videos.

📘 How to Use

  1. Select a folder using the input or drag & drop it.
  2. Use the buttons to filter HTML tags by file type (CSS, JS, images, etc).
  3. Click a file in the folder tree to preview its HTML tag.
  4. CTRL+Click a file to preview it (if supported).
  5. Click "Copy All" to copy generated tags to clipboard.
